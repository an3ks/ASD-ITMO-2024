## Запуск и тестирование

1. **Клонирование репозитория:**
   ```bash
   git clone https://github.com/an3ks/ASD-ITMO-2024
   ```
2. **Запуск программы:**
   Перейдите в папку с задачей и выполните:
   ```bash
   python src/ex1.py
   ```
3. **Запуск тестов:**
   Перейдите в папку lab 1 и выполните:
   ```bash
   python runAllSrc.py
   ```
